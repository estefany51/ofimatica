Sitios|Migración|Para hacer una correcta migración práctica, lo que debemos hacer es lo siguiente: \n* Crearemos una nueva base de datos y usuarios quien tendrá todos los permisos sobre la base de datos.\n\nRECORDAR \ncreate database basenueva;\ncreate user usuario identified by &#039;123abc&#039;;\ngrant all privileges on basenueva.* to usuario;\n\nposterior a la creación de nuestra base de datos nos posicionaremos sobre la ruta donde se encuentra la base vieja y ejecutaremos el siguiente comando\n\nmysql -u usuario -p basenueva &lt; basevieja.sql (al presionar ENTER nos pedirá la contraseña)\npasswd: 123abc (se ingresará la contraseña del nuevo usuario)|
